# Web App

## Role

React/Vite web client for audience/admin pages and Sprint 3 checker fallback.

## Current Notes

- `/checker` is the mobile web fallback for online check-in demo.
- Checker page supports token entry, concert/gate/device ids, preload, manual QR payload scan, and guest scan by id or phone.
- Full offline SQLite sync is not implemented in web; that remains Sprint 4/mobile scope.
